# 🎬 IMDB Movie Review Sentiment Analysis (SimpleRNN)

This project is an end-to-end Machine Learning application that classifies movie reviews as **Positive** or **Negative** using a Deep Learning model built with SimpleRNN.

---

## 🚀 Features

* 🔍 Sentiment classification of movie reviews
* 🧠 Deep Learning model using SimpleRNN
* ⚡ Interactive interface using Streamlit
* 📊 Displays prediction confidence score

---

## 🛠 Tech Stack

* **Language:** Python
* **Libraries:** TensorFlow, Keras, NumPy
* **Dataset:** IMDB Movie Reviews
* **Frontend:** Streamlit

---

## 🧠 Model Details

* Model Type: SimpleRNN
* Embedding layer for word representation
* Trained on IMDB dataset
* Binary classification (Positive / Negative)
* Achieved ~80–85% validation accuracy
